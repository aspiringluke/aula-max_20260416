# Requisitos Funcionais | Classes Candidatas

> Em <span style="color: rgb(255,200,0);">amarelo</span>, os substantivos das <span style="color: rgb(80,235,0);">classes candidatas</span>

- **RF01 — Cadastro de <span style="color: rgb(255,200,0);">Alunos</span>**<br>
O sistema deve permitir cadastrar novos alunos contendo dados pessoais, contato, endereço e plano contratado.

- **RF02 — Gerenciamento de <span style="color: rgb(255,200,0);">Planos</span>**<br>
O sistema deve permitir criar, editar, ativar e desativar tipos de planos (mensal, trimestral, anual, musculação, funcional etc.).

- **RF03 — Controle de <span style="color: rgb(255,200,0);">Pagamentos</span>**<br>
O sistema deve registrar pagamentos realizados na <span style="color: rgb(255,200,0);">recepção</span> (dinheiro, cartão ou PIX) e gerar boletos/cartões para pagamentos online.

- **RF04 — Regularidade do Aluno**<br>
O sistema deve verificar automaticamente se o aluno está com mensalidade em dia.

- **RF05 — Controle de <span style="color: rgb(255,200,0);">Acesso</span>**<br>
O sistema deve se integrar à catraca para validar a entrada do aluno por RFID.

- **RF06 — <span style="color: rgb(255,200,0);">Agendamento</span> de <span style="color: rgb(255,200,0);">Aulas</span>**<br>
O aluno deve poder visualizar horários e reservar vagas nas aulas disponíveis.

- **RF07 — Lista de <span style="color: rgb(255,200,0);">Presença</span>**<br>
Instrutores devem registrar presença dos alunos nas aulas.

- **RF08 — <span style="color: rgb(255,200,0);">Avaliação Física</span>**<br>
Instrutores devem registrar avaliações físicas (peso, IMC, percentual de gordura etc.) e anexar arquivos se necessário.

- **RF09 — Relatórios Gerenciais**<br>
O <span style="color: rgb(255,200,0);">gerente</span> deve poder emitir relatórios como:
    - inadimplência
    - alunos ativos
    - histórico de acessos
    - ocupação das aulas

- **RF10 — <span style="color: rgb(255,200,0);">Notificações</span>**<br>
O sistema deve enviar notificações ao aluno sobre:
    - vencimento de mensalidade
    - confirmação de agendamento
    - liberação de nova avaliação física
